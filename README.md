# Dashboards · Javier Forero

Colección de **dashboards interactivos** que no solo grafican: interpretan, diagnostican y convierten los datos en decisión. Cada uno es autónomo (un solo HTML, datos embebidos) y aplica el método de [javierforero.co](https://javierforero.co).

## Cómo agregar un dashboard nuevo
1. Crea una carpeta hermana de `sla-operaciones/` (p. ej. `inventario/`) con su propio `index.html`.
2. Duplica una tarjeta en el `index.html` del hub, cambia título, descripción y el enlace a la nueva carpeta.
3. Commit + push. GitHub Pages lo publica en `https://jaforero.github.io/dashboards/<carpeta>/`.

## Publicar (GitHub Pages)
Settings → Pages → *Deploy from a branch* → `main` / `/ (root)`. El `.nojekyll` ya está incluido.
